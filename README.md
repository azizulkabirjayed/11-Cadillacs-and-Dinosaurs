# Features
* 3D OpenGL action game built with Python & PyOpenGL
* Multiple enemy types & boss fight
* Zone-based progression system
* Punch, jump attack & spinning kick combat
* Player health, scoring & collectibles
* Obstacles (trees, cars) with collision detection
* Day–night cycle with Snow particle effects
* Third-person & first-person camera modes & camera movement
* Cheat mode with bomb attacks
* Pause, restart & in-game UI controls

# Controls
| Key | Action |
|----|-------|
| W | Move Forward|
| A | Move Backward|
| S | Move Left|
| D | Move Right|
| Left Mouse Button | Punch |
| Space | Jump |
| Space+Left Mouse Button | Jump Attack |
| Space+Right Mouse Button | Spinning Kick |
| C | Toggle Cheat Mode |
| Q | Toggle First-Person/Third-Person Camera |
| →/ ← | Rotate Third-Person Camera Left/ Right |
| ↑/ ↓ | Rotate Third-Person Camera Up/ Down |
| R | Restart |
